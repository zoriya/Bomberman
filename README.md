## Run (compile from the sources)

Clone the project

```bash
git clone https://github.com/AnonymusRaccoon/Bomberman.git
```

Go to the project directory

```bash
cd Bomberman
```

Create `build` dir and compile using CMake

```bash
mkdir build && cd build && cmake .. && make
```

Enjoy !

```bash
./bomberman
```
